### 피그마 사용방법 익히기

### 피그마 : 화면 기획 협업 툴

```
-- -- -- ---------------------<  피그마 단축키 정리 >> ---------------------------------- -- -- --
-- 참고 페이지 : https://notepad96.tistory.com/279
Figma 단축키

Figma에서 사용 가능한 단축키들은 Figma 내에서 " Ctrl + Shift + / "를 입력하면 아래의 이미지 같이 각 영역별로 확인이 가능하다.

1. Essential
단축키	기능
Ctrl + \	메뉴창 UI 보이기/감추기
I (대문자 i)	색상 추출 스포이드
Ctrl + /	Figma 기능 검색
 

2. Tools
단축키	기능
V	이동 
F	프레임 
P	펜
Shift + P	연필
T	텍스트
R	사각형
O	원형
L	라인
Shift + L	화살표
C	코멘트 추가
S	슬라이스
 

3. View
단축키	기능
CTRL + Alt + \	멀티플레이어(동시 작업자) 커서 보이기/감추기
Shift + R	눈금자
Shift + O	아웃라인 보이기/숨기기
Shift + G	레이아웃 그리드 보이기/숨기기
Alt + 1	Layers 창
Alt + 2	Assets 창
Alt + 3	팀 라이브러리
Alt + 8	Design 창
Alt + 9	Prototype 창
Alt + 0	Inspect 창
 

4. Zoom
단축키	기능
Space + 드래그	메뉴창 UI 보이기/감추기
+	화면 확대
-	화면 축소
Ctrl + 0	화면 비율 100%
Shift + 1	화면 모니터 비율에 맞춤
Shift + 2	선택한 요소 기준으로 비율 맞춤
N	프레임 기준 화면 맞춤(다음 프레임으로 이동)
Shift N	프레임 기준 화면 맞춤(이전 프레임으로 이동)
Page Up	이전 페이지로 이동
Page Down	다음 페이지로 이동
Home	이전 프레임으로 이동
End	다음 프레임으로 이동
 

5. Text
단축키	기능
Ctrl + B	굵게(Bold)
Ctrl + U	밑줄
Ctrl + K	링크 걸기
Ctrl + Art + L	텍스트 왼쪽 정렬
Ctrl + Art + T	텍스트 가운데 정렬
Ctrl + Art + R	텍스트 오른쪽 정렬
Ctrl + Art + J	텍스트 가로 맞춤 정렬
Ctrl + Shift + < , >	폰트 사이즈 조절
Ctrl + Alt + < , >	폰트 굵기 조절
Alt + < , >	텍스트 간격 조절
Alt + Shift + < , >	텍스트 줄 간격 조절
 

6. Shape
단축키	기능
B	페인트 버킷
Alt + /	Fill 제거 (채우기)
Shift + /	Stroke 제거 (테두리)
Shift + X	Fill과 Stroke 교환
 

7. Selection
단축키	기능
Ctrl + A	전체 선택
Ctrl + Shift + A	선택되지 않은 요소들 전체 선택
Esc	선택 취소
Ctrl + G	그룹 지정
Ctrl + Shift + G	그룹 해제
Ctrl + Alt + G	선택 요소 프레임으로 만들기
Ctrl + Shift + H	선택 요소 보이기/감추기
Ctrl + Shigt + L	선택 요소 잠금/해제
 

8. Cursor
단축키	기능
Alt + Pointing	요소간 거리 측정
Alt + Moving	선택 요소 복사
Space	끌어서 화면 이동
Alt + Resizing	요소 중심을 기준으로 크기 조절
Shift + Resizing	요소 비율을 유지하며 크기 조절
 

9. Edit
단축키	기능
Ctrl + D	복제 및 반복
Ctrl + R	이름 변경
Ctrl + Alt + C	속성 복사
Ctrl + Alt + V	속성 붙여넣기
 

 

10. Transform
단축키	기능
Shift + H	수평으로 뒤집기 (좌우반전)
Shift + V	수직으로 뒤집기 (상하반전)
Ctrl + Alt + M	마스크 사용
Ctrl + Shift + K	이미지 불러내기
Enter	Shape 혹은 Image 수정 및 완료
0 0	투명도 0%
1	투명도 10%
5	투명도 50%
0	투명도 100%
 

11. Arrange
단축키	기능
Ctrl + ]	앞으로 이동
Ctrl + [	뒤로 이동
]	맨 앞으로 이동
[	맨 뒤로 이동
Alt + A , D	왼쪽, 오른쪽 정렬
Alt + W , S	상단, 하단 정렬
Alt + H , V	수평, 수직 정렬
 

12. Components
단축키	기능
Alt + 3	팀 라이브러리
Ctrl + Alt + K	컴포넌트 생성
Shift + I	리소스 삽입(컴포넌트, 플러그인)
```

- 메인 프로젝트 : 헬로 키즈 앱(가칭)
- 서브 프로젝트 : 헬로 컬처 앱(가칭)
